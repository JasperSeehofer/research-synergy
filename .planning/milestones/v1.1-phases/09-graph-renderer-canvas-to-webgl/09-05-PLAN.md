    **WebGL2Renderer struct:**
    ```rust
    pub struct WebGL2Renderer {
        gl: WebGl2RenderingContext,
        node_program: WebGlProgram,
        edge_program: WebGlProgram,
        node_vao: WebGlVertexArrayObject,
        edge_vao: WebGlVertexArrayObject,
        width: u32,
        height: u32,
    }
    ```

    **Constructor `WebGL2Renderer::new(canvas: &HtmlCanvasElement) -> Self`:**
    - Acquire `webgl2` context
    - Compile and link two shader programs: one for nodes (instanced circles), one for edges (lines)
    - Create VAOs for each

    **Node shader program (instanced quads):**

    Vertex shader:
    ```glsl
    #version 300 es
    in vec2 a_quad;
    in vec2 a_position;
    in float a_radius;
    in float a_alpha;
    in vec3 a_color;

    uniform vec2 u_resolution;
    uniform vec2 u_offset;
    uniform float u_scale;

    out vec2 v_local;
    out float v_alpha;
    out vec3 v_color;

    void main() {
        v_local = a_quad;
        v_alpha = a_alpha;
        v_color = a_color;
        vec2 world = a_position + a_quad * a_radius;
        vec2 screen = (world * u_scale + u_offset) / u_resolution * 2.0 - 1.0;
        gl_Position = vec4(screen.x, -screen.y, 0.0, 1.0);
    }
    ```

    Fragment shader:
    ```glsl
    #version 300 es
    precision mediump float;
    in vec2 v_local;
    in float v_alpha;
    in vec3 v_color;
    out vec4 fragColor;

    void main() {
        float d = length(v_local);
        if (d > 1.0) discard;
        // Smooth edge for anti-aliasing
        float edge = 1.0 - smoothstep(0.9, 1.0, d);
        fragColor = vec4(v_color, v_alpha * edge);
    }
    ```

    **Edge shader program (lines):**

    Vertex shader:
    ```glsl
    #version 300 es
    in vec2 a_position;
    in vec3 a_color;
    in float a_alpha;

    uniform vec2 u_resolution;
    uniform vec2 u_offset;
    uniform float u_scale;

    out vec3 v_color;
    out float v_alpha;

    void main() {
        vec2 screen = (a_position * u_scale + u_offset) / u_resolution * 2.0 - 1.0;
        gl_Position = vec4(screen.x, -screen.y, 0.0, 1.0);
        v_color = a_color;
        v_alpha = a_alpha;
    }
    ```

    Fragment shader:
    ```glsl
    #version 300 es
    precision mediump float;
    in vec3 v_color;
    in float v_alpha;
    out vec4 fragColor;

    void main() {
        fragColor = vec4(v_color, v_alpha);
    }
    ```

    **`draw` implementation:**
    1. Clear with `#0d1117` background: `gl.clear_color(0.051, 0.067, 0.09, 1.0)`, `gl.clear(COLOR_BUFFER_BIT)`
    2. Enable blending: `gl.enable(BLEND)`, `gl.blend_func(SRC_ALPHA, ONE_MINUS_SRC_ALPHA)`
    3. Set uniforms: u_resolution, u_offset (from viewport), u_scale (from viewport)
    4. **Draw edges:** Build vertex buffer with 2 vertices per visible edge (position, color, alpha). Use `gl.draw_arrays(LINES, 0, vertex_count)`. Colors: `#404040` alpha 0.35 for regular, `#f85149` alpha 1.0 for contradiction, `#d29922` alpha 1.0 for bridge. Respect show_contradictions/show_bridges flags.
    5. **Draw nodes:** Build per-instance buffers (position, radius, alpha, color). Quad vertices: [(-1,-1), (1,-1), (1,1), (-1,1)]. Use `gl.draw_arrays_instanced(TRIANGLE_FAN, 0, 4, node_count)` with `vertex_attrib_divisor` on per-instance attributes. Node colors: `#4a9eff` (default), `#58a6ff` (hovered/selected), `#2a3a4f` (dimmed).
    6. Unbind VAO: `gl.bind_vertex_array(None)`

    **Note on labels:** WebGL2 does not natively render text. For the WebGL renderer, node labels are drawn as a Canvas 2D overlay on a second canvas stacked on top. The WebGL2Renderer should expose a `pub fn node_screen_positions(&self, state: &GraphState, viewport: &Viewport) -> Vec<(f64, f64, String)>` method that returns screen-space positions and labels for the GraphPage to draw on a 2D overlay canvas. This is the standard approach for WebGL text.

    **Arrowheads in WebGL:** Draw as small triangles at each edge target. Build triangle vertex buffer for arrowheads as part of the edge draw pass (3 vertices per arrowhead, same color as parent edge).

    **Helper functions:**
    - `compile_shader(gl, shader_type, source) -> WebGlShader`
    - `link_program(gl, vert, frag) -> WebGlProgram`
    - `create_buffer(gl, data: &[f32]) -> WebGlBuffer`
    - `hex_to_rgb(hex: &str) -> (f32, f32, f32)` for color conversion

    **`resize` implementation:** `gl.viewport(0, 0, width, height)`, update stored dimensions.

    3. **Add `make_renderer` factory function** to `resyn-app/src/graph/renderer.rs` (or a new `resyn-app/src/graph/mod.rs` function):
    ```rust
    pub fn make_renderer(canvas: &web_sys::HtmlCanvasElement, node_count: usize) -> Box<dyn Renderer> {
        // Probe WebGL2 on a temporary canvas (avoids contaminating main canvas)
        let document = web_sys::window().unwrap().document().unwrap();
        let probe: web_sys::HtmlCanvasElement = document
            .create_element("canvas").unwrap()
            .dyn_into().unwrap();
        let can_use_webgl = probe.get_context("webgl2").ok().flatten().is_some();

        if node_count > WEBGL_THRESHOLD && can_use_webgl {
            Box::new(WebGL2Renderer::new(canvas))
        } else {
            if node_count > WEBGL_THRESHOLD {
                web_sys::console::warn_1(&"WebGL2 unavailable, falling back to Canvas 2D".into());
            }
            Box::new(Canvas2DRenderer::new(canvas))
        }
    }
    ```
