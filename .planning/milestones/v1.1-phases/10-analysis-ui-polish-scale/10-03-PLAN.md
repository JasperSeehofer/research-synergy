1. In `resyn-app/src/app.rs`:
   - Change `SelectedPaper` to carry both paper ID and optional provenance context:
     ```rust
     #[derive(Clone, Debug, Default)]
     pub struct DrawerOpenRequest {
         pub paper_id: String,
         pub initial_tab: DrawerTab,
         pub highlight_snippet: Option<String>,
         pub highlight_section: Option<String>,
     }

     #[derive(Clone, Debug, Default, PartialEq)]
     pub enum DrawerTab {
         #[default]
         Overview,
         Source,
     }

     #[derive(Clone)]
     pub struct SelectedPaper(pub RwSignal<Option<DrawerOpenRequest>>);
     ```
   - Update all existing uses of `SelectedPaper` — anywhere that does `selected.set(Some(id_string))` must now use `selected.set(Some(DrawerOpenRequest { paper_id: id_string, initial_tab: DrawerTab::Overview, highlight_snippet: None, highlight_section: None }))`.
   - Search for all files that use `SelectedPaper` and update them. Key locations:
     - `resyn-app/src/layout/drawer.rs` — reads selected paper ID
     - `resyn-app/src/components/gap_card.rs` — sets selected paper
     - `resyn-app/src/pages/graph.rs` — sets selected paper on node click
     - `resyn-app/src/pages/papers.rs` — sets selected paper on row click (if exists)

   - In `DrawerContent`, add tab state:

     ```rust
         <button

     </div>
     ```
   - Below the tab strip, conditionally render:
     - If `active_tab == Overview`: existing drawer body content (abstract, methods, findings, open problems)
     - If `active_tab == Source`: new `SourceTabBody` component

   - Create `SourceTabBody` component:
     ```rust

         extraction: Option<TextExtractionResult>,
         highlight_snippet: Option<String>,
         highlight_section: Option<String>,
     ) -> impl IntoView
     ```
     - If `extraction` is None: show empty state "Not yet analyzed" / "Run analysis on this paper..."
     - If `extraction.extraction_method == AbstractOnly`: show abstract text + label "Abstract only -- full text unavailable"
     - If extraction has sections: iterate each section (ABSTRACT, INTRODUCTION, METHODS, RESULTS, CONCLUSION), for each:
       - Show section header (uppercase, 12px label, `drawer-section-title` class)
       - Show section text. If `highlight_section` matches this section name and `highlight_snippet` is Some, use `find_highlight_range` to locate the snippet and split text into three parts:
         - before-highlight (plain text)
         - highlight span (`<mark class="snippet-highlight">{matched_text}</mark>`)
         - after-highlight (plain text)
       - If no match found for snippet, render section text plain with a note "Snippet not found in section text"
     - Import `resyn_core::analysis::highlight::find_highlight_range` (feature-gated check: this is pure Rust, no ssr gate, should be available in WASM)
