## What Was Built

**PaletteEntry type** — `pub struct PaletteEntry { keyword, r, g, b, slot_index }` with Serialize/Deserialize derives. Added as `#[serde(default)]` field on `GraphData` for backward compatibility (absent field deserializes as empty vec).

**SurrealDB migration 8** — `keyword_palette` SCHEMAFULL table with keyword (UNIQUE index), r/g/b/slot_index (int), corpus_fingerprint (string), computed_at (string). Follows existing migration pattern in `apply_migration_8()`.

**PaletteRepository** — Four methods: `upsert_palette()` (full DELETE + re-insert for atomic replacement), `get_palette()` (ordered by slot_index), `get_corpus_fingerprint()` (reads fingerprint from any entry), `clear_palette()`.

**TF-IDF variance computation** — In `get_graph_data()`, computes inter-paper keyword variance from `analysis_map`. Fingerprint = `paper_count:N`. Cache hit if fingerprint matches; recompute on corpus change. Top-8 keywords by variance get assigned pre-computed OKLCH RGB values from 8-slot constant table. DB write is fire-and-forget.

**GraphState extensions** — `palette: Vec<PaletteEntry>` propagated from `GraphData.palette`. `NodeState.topic_dimmed: bool` initialized to `false` for all nodes.
