    - Test 3: ExtractionRepository::upsert_extraction stores a TextExtractionResult and get_extraction retrieves it
    - Test 4: ExtractionRepository::extraction_exists returns false for missing, true after upsert

    - Test 6: get_all_extractions returns all stored extraction records

  <action>
    **schema.rs** -- Replace `init_schema()` with `migrate_schema()`:
    1. Create `migrate_schema(db: &Surreal<Any>) -> Result<(), ResynError>` that:
       - Defines `schema_migrations` table (SCHEMAFULL) with `version: int` and `applied_at: string` fields using `IF NOT EXISTS`
       - Reads current version via `get_schema_version()` (query `SELECT version FROM schema_migrations ORDER BY version DESC LIMIT 1`, default 0)
       - If version < 1: runs `apply_migration_1()` -- the existing paper + cites DDL (moved verbatim from current `init_schema`)
       - If version < 2: runs `apply_migration_2()` -- text_extraction table DDL:
         ```sql
         DEFINE TABLE IF NOT EXISTS text_extraction SCHEMAFULL;
         DEFINE FIELD IF NOT EXISTS arxiv_id ON text_extraction TYPE string;
         DEFINE FIELD IF NOT EXISTS extraction_method ON text_extraction TYPE string;
         DEFINE FIELD IF NOT EXISTS abstract_text ON text_extraction TYPE option<string>;
         DEFINE FIELD IF NOT EXISTS introduction ON text_extraction TYPE option<string>;
         DEFINE FIELD IF NOT EXISTS methods ON text_extraction TYPE option<string>;
         DEFINE FIELD IF NOT EXISTS results ON text_extraction TYPE option<string>;
         DEFINE FIELD IF NOT EXISTS conclusion ON text_extraction TYPE option<string>;
         DEFINE FIELD IF NOT EXISTS is_partial ON text_extraction TYPE bool;
         DEFINE FIELD IF NOT EXISTS extracted_at ON text_extraction TYPE string;
         DEFINE INDEX IF NOT EXISTS idx_extraction_arxiv_id ON text_extraction FIELDS arxiv_id UNIQUE;
         ```
       - Each `apply_migration_N` runs idempotent DDL first, then inserts version record: `CREATE schema_migrations CONTENT { version: N, applied_at: $now }`
    2. Create helper `get_schema_version()` and `set_schema_version()` private functions.
    3. Delete the old `init_schema()` function.

    **client.rs** -- Update `setup()` to call `migrate_schema(db).await?` instead of `init_schema(db).await?`. Update the import.

    **queries.rs** -- Add `ExtractionRepository`:
    1. Create `ExtractionRecord` struct with SurrealValue derive (mirrors PaperRecord pattern) with fields matching the text_extraction table schema. The `extraction_method` field is stored as a string ("AbstractOnly" or "Ar5ivHtml").
    2. Add `impl From<&TextExtractionResult> for ExtractionRecord` and `impl ExtractionRecord { fn to_extraction_result(&self) -> TextExtractionResult }`.
    3. Create `fn extraction_record_id(arxiv_id: &str) -> RecordId` using `RecordId::new("text_extraction", strip_version_suffix(arxiv_id))`.
    4. Create `pub struct ExtractionRepository<'a> { db: &'a Db }` with:
       - `pub fn new(db: &'a Db) -> Self`
       - `pub async fn upsert_extraction(&self, result: &TextExtractionResult) -> Result<(), ResynError>` -- uses UPSERT pattern from PaperRepository
       - `pub async fn get_extraction(&self, arxiv_id: &str) -> Result<Option<TextExtractionResult>, ResynError>` -- select by record ID
       - `pub async fn extraction_exists(&self, arxiv_id: &str) -> Result<bool, ResynError>`
       - `pub async fn get_all_extractions(&self) -> Result<Vec<TextExtractionResult>, ResynError>`

    Note: Use `strip_version_suffix()` consistently in all record ID construction, matching the PaperRepository pattern.
  </action>
