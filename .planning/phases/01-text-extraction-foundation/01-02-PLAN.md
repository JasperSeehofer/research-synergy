    Create `src/data_aggregation/text_extractor.rs` with:

    1. **`Ar5ivExtractor` struct** mirroring `ArxivHTMLDownloader` pattern:
       - Fields: `last_called: Option<Instant>`, `call_per_duration: Duration`, `client: reqwest::Client`
       - `pub fn new(client: reqwest::Client) -> Self` with default 3s rate limit
       - `pub fn with_rate_limit(mut self, duration: Duration) -> Self`
       - `async fn rate_limit_check(&mut self)` -- copy from ArxivHTMLDownloader

    2. **`fn ar5iv_url(arxiv_id: &str) -> String`** -- `format!("https://arxiv.org/html/{}", arxiv_id)`. Use `paper.id` directly (not pdf_url conversion). Works for both arXiv and InspireHEP-sourced papers.

    3. **`pub async fn extract(&mut self, paper: &Paper) -> TextExtractionResult`**:
       - Call `rate_limit_check().await`
       - Build URL from `strip_version_suffix(&paper.id)`
       - Send GET request, check `response.status().is_success()`
       - On non-200 or network error: return `TextExtractionResult::from_abstract(paper)` (immediate fallback, no retry per user decision)
       - On 200: parse HTML with `scraper::Html::parse_document()`, call `parse_sections()`, build `TextExtractionResult` with `ExtractionMethod::Ar5ivHtml`
       - Set `is_partial = false` only when extraction_method is Ar5ivHtml (even if some sections are None -- best-effort is fine)

    4. **`fn parse_sections(html: &Html) -> SectionMap`**:
       - Abstract: CSS selector `.ltx_abstract` -- extract text, strip leading "Abstract" word if present
       - Named sections: selector `section.ltx_section, div.ltx_section`
       - Title detection: selector `.ltx_title_section, .ltx_title_chapter` within each section
       - Content: selector `.ltx_para` within each section, collect text
       - **Title normalization**: strip leading whitespace, digits, Roman numerals (I, II, III, IV, V, VI, VII, VIII, IX, X), dots, colons. Convert to lowercase. Then match with `contains()`:
         - introduction: "introduction", "motivation"
         - methods: "method", "approach", "model", "framework", "formalism", "setup"
         - results: "result", "discussion", "experiment", "analysis", "observation", "finding"
         - conclusion: "conclusion", "summary", "outlook"
       - **Exclusions**: Skip sections whose normalized title contains "reference", "bibliography", "acknowledgement", "appendix"
       - If multiple sections match the same category, concatenate with newline separator

    5. **Text cleaning**: Trim whitespace from each section. If a section's text is empty after trimming, set it to None.

    Add `pub mod text_extractor;` to `src/data_aggregation/mod.rs`.

       #[arg(long, default_value_t = false)]

       #[arg(long, default_value_t = false)]

    2. Add analysis pipeline step after the "Persist to database" block (before `launch_visualization`). This step also runs in `db_only` mode when `--analyze` is specified:
       ```rust
       if cli.analyze {
           let Some(ref db) = db else {
               error!("--analyze requires --db to be specified");
               std::process::exit(1);
           };
           let extraction_repo = database::queries::ExtractionRepository::new(db);
           let paper_repo = database::queries::PaperRepository::new(db);
           let all_papers = paper_repo.get_all_papers().await.unwrap_or_else(|e| {
               error!(error = %e, "Failed to load papers for analysis");
               std::process::exit(1);
           });

           let client = utils::create_http_client();
           let mut extractor = data_aggregation::text_extractor::Ar5ivExtractor::new(client)
               .with_rate_limit(std::time::Duration::from_secs(cli.rate_limit_secs));

           let mut abstract_only_count: usize = 0;
           let mut skipped_count: usize = 0;
           for paper in &all_papers {
               let stripped_id = utils::strip_version_suffix(&paper.id);
               if extraction_repo.extraction_exists(&stripped_id).await.unwrap_or(false) {
                   skipped_count += 1;
                   continue;
               }
               let result = if cli.skip_fulltext {
                   datamodels::extraction::TextExtractionResult::from_abstract(paper)
               } else {
                   extractor.extract(paper).await
               };
               if result.is_partial {
                   abstract_only_count += 1;
               }
               if let Err(e) = extraction_repo.upsert_extraction(&result).await {
                   error!(paper_id = paper.id, error = %e, "Failed to store extraction");
               }
           }

           let analyzed = all_papers.len() - skipped_count;
           info!(
               abstract_only = abstract_only_count,
               analyzed = analyzed,
               skipped = skipped_count,
               total = all_papers.len(),
               "{}/{} papers used abstract-only extraction ({} skipped, already cached)",
               abstract_only_count, analyzed, skipped_count
           );
       }
       ```

Restructure: after the `db_only` early-return block and after the normal crawl+persist block, add the shared analysis step.
