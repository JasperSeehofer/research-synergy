**GapCard component (resyn-app/src/components/gap_card.rs):**

Create a `GapCard` component that accepts a `GapFinding` prop:

```rust
#[component]
pub fn GapCard(finding: GapFinding) -> impl IntoView { ... }
```

1. **Type badge**: `span class="badge badge-contradiction"` or `"badge badge-bridge"` based on `finding.gap_type`. Text: "Contradiction" or "ABC-Bridge".

2. **Paper title links**: For each `paper_id` in `finding.paper_ids`, render a clickable link (`span` with `class="paper-link"` styled as accent color, cursor pointer). On click, call `set_selected_paper.set(Some(paper_id.clone()))` to open the paper drawer. Display the paper ID as text (the actual title is not in GapFinding â€” show the arXiv ID as the link text).

3. **Shared terms as tags**: For each term in `finding.shared_terms`, render `span class="tag"` with the term text.

4. **Confidence bar**: A horizontal bar showing confidence level. The bar container is `div class="confidence-bar"`. The fill is `div class="confidence-bar-fill"` with:
   - `style` attribute setting `width` to `{confidence * 100}%`
   - `style` attribute setting `background` to a color interpolated from the gradient: red (#f85149) at 0%, yellow (#d29922) at 50%, green (#3fb950) at 100%. Use a CSS linear-gradient approach: set `background: linear-gradient(90deg, #f85149, #d29922, #3fb950); background-size: {100 / confidence}% 100%;` OR simpler: compute the color in Rust based on confidence value:
     - confidence < 0.4: `#f85149`
     - 0.4 <= confidence < 0.8: `#d29922`
     - confidence >= 0.8: `#3fb950`
   - Beside the bar, a `span class="confidence-label"` showing `"{N}%"` where N = `(confidence * 100.0) as u32`.
   - `title` attribute on bar: `"Confidence: {N}%"` per UI-SPEC copywriting.

5. **Expand/collapse for justification**: A chevron button at the bottom of the card. Use a local `RwSignal<bool>` for expanded state. When collapsed, `div class="gap-justification"` (max-height: 0). When expanded, `div class="gap-justification expanded"`. The justification text is `finding.justification` displayed in Body (14px) muted color.

6. Wrap everything in `div class="card"` with margin-bottom `var(--space-lg)` (24px gap between cards per UI-SPEC).

Add `paper-link` CSS to `main.css` if not already present: `.paper-link { color: var(--color-accent); cursor: pointer; }`, `.paper-link:hover { text-decoration: underline; }`.
