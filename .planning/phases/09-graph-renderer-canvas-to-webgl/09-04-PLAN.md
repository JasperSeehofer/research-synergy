    **Shared state struct** (held in `Rc<RefCell<...>>`, outside Leptos reactive graph per RESEARCH anti-patterns):
    ```rust
    struct RenderState {
        graph: GraphState,
        viewport: Viewport,
        renderer: Box<dyn Renderer>,
        interaction: InteractionState,
        worker: Option<WorkerBridge>,
    }
    ```

    **GraphPage component:**
    - Create `canvas_ref = NodeRef::<Canvas>::new()`
    - Create Leptos signals for overlay state: `show_contradictions: RwSignal<bool>` (true), `show_bridges: RwSignal<bool>` (true), `simulation_running: RwSignal<bool>` (true)
    - Create `tooltip_signal: RwSignal<Option<TooltipData>>` where `TooltipData { text: String, x: f64, y: f64 }`
    - Fetch graph data: `let graph_resource = Resource::new(|| (), |_| get_graph_data());`

    **Effect (fires on mount + when graph_resource resolves):**
    ```rust
    Effect::new(move |_| {
        let Some(canvas) = canvas_ref.get() else { return };
        let canvas: web_sys::HtmlCanvasElement = canvas.into();
        let Some(Ok(data)) = graph_resource.get() else { return };

        // Size canvas to container
        let width = canvas.offset_width() as u32;
        let height = canvas.offset_height() as u32;
        canvas.set_width(width);
        canvas.set_height(height);

        let renderer = Canvas2DRenderer::new(&canvas);
        let viewport = Viewport::new(width as f64, height as f64);
        let graph_state = GraphState::from_graph_data(data);

        let state = Rc::new(RefCell::new(RenderState {
            graph: graph_state,
            viewport,
            renderer: Box::new(renderer),
            interaction: InteractionState::Idle,
            worker: None,
        }));

        // Spawn worker and send initial layout
        let state_for_worker = state.clone();
        let worker = WorkerBridge::new(move |output| {
            let mut s = state_for_worker.borrow_mut();
            for (i, (x, y)) in output.positions.into_iter().enumerate() {
                if i < s.graph.nodes.len() && !s.graph.nodes[i].pinned {
                    s.graph.nodes[i].x = x;
                    s.graph.nodes[i].y = y;
                }
            }
            if output.converged {
                s.graph.simulation_running = false;
            }
        });
        state.borrow_mut().worker = Some(worker);

        // Attach event listeners (mouse/pointer/wheel)
        attach_event_listeners(&canvas, state.clone(), tooltip_signal, show_contradictions, show_bridges);

        // Start RAF render loop
        let handle = start_render_loop(canvas.clone(), state.clone(), simulation_running);
        on_cleanup(move || handle.cancel());
    });
    ```

    **RAF render loop** (`start_render_loop`):
    Follow the `Rc<RefCell<Option<Closure<dyn FnMut()>>>>` pattern from RESEARCH Pattern 2. Each frame:
    1. Borrow state
    2. Sync show_contradictions/show_bridges signals into graph state
    3. If simulation_running, send LayoutInput to worker (nodes as NodeData vec, edges as index pairs, ticks: 1)
    4. Call `renderer.draw(&graph, &viewport)`
    5. Schedule next frame via `request_animation_frame`
    Return `RafHandle` with cancel support.

    **Event listeners** (`attach_event_listeners`):
    Use `Closure::new` for each event, store closures to prevent drop:

    - **mousemove:** Convert to canvas coords via `get_bounding_client_rect()`. Convert to world via `viewport.screen_to_world()`. Check `find_node_at` — if hit, update `hovered_node`, set tooltip. Check `find_edge_at(threshold: 4.0)` — if hit, set edge tooltip. If InteractionState::Panning, update viewport offset. If InteractionState::DraggingNode, update node position.

    - **mousedown:** Convert coords. If `find_node_at` hits, check if node is already pinned — if so, set InteractionState::DraggingNode. If node not pinned, pin it and start DraggingNode. If no node hit, start Panning (store start coords + viewport offset).

    - **mouseup:** If DraggingNode, return to Idle (node stays pinned). If Panning, return to Idle. If click (no significant movement), check `find_node_at` — if hit pinned node that wasn't dragged, unpin it. If hit unpinned node, set `SelectedPaper` context to open drawer. If no node, deselect (set selected_node = None).

    - **dblclick:** Reset viewport to initial centered position (scale 1.0, offset = canvas_size / 2).

    - **wheel:** Call `zoom_toward_cursor` with delta from `event.delta_y()`.

    - **pointerleave:** Clear tooltip, clear hovered_node/hovered_edge.

    **Tooltip rendering** in the view:
    ```rust
    {move || tooltip_signal.get().map(|t| view! {
        <div class="graph-tooltip" style=format!("left: {}px; top: {}px;", t.x + 12.0, t.y - 12.0)>

        </div>
    })}
    ```
