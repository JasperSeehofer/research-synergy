## Spacing Scale

All tokens pre-exist in `main.css` `:root`. Phase 9 uses these values as-is.

| Token | Value | Usage |
|-------|-------|-------|
| `--space-xs` | 4px | Icon gaps, overlay control padding |
| `--space-sm` | 8px | Compact element spacing, tooltip padding |
| `--space-md` | 16px | Overlay panel internal padding |
| `--space-lg` | 24px | Overlay panel gaps |
| `--space-xl` | 32px | Layout gaps |
| `--space-2xl` | 48px | Major section breaks |
| `--space-3xl` | 64px | Page-level spacing (empty state) |

**Exceptions:**

- Graph canvas: zero padding — occupies 100% of `.content-area` minus sidebar; canvas element must fill its container completely (`width: 100%; height: 100%`) so no spacing tokens apply to the canvas itself
- Overlay controls: use 8px internal padding (sm) and 8px gap between control groups
- Node tooltip: 4px vertical / 8px horizontal padding (xs/sm) — compact, near-cursor

**Source:** Pre-existing tokens from `main.css`. No new spacing values introduced.

## Typography

All type roles pre-exist in `main.css`. Phase 9 uses the same scale.

| Role | Size | Weight | Line Height | CSS Variable |
|------|------|--------|-------------|--------------|
| Label | 12px | 400 | 1.3 | `--font-size-label` / `--font-weight-regular` |
| Body | 14px | 400 | 1.5 | `--font-size-body` / `--font-weight-regular` |
| Heading | 18px | 600 | 1.3 | `--font-size-heading` / `--font-weight-semibold` |
| Display | 28px | 400 | 1.2 | `--font-size-display` / `--font-weight-regular` |

**Graph-specific typography rules:**

- **Node labels** (Canvas 2D and WebGL2): 11px monospace — compact, readable at moderate zoom; renders in Canvas 2D as `ctx.set_font("11px monospace")`, color `#cccccc` (matches `--color-text-muted`). Labels display "First Author Year" format (e.g., "Einstein 2024").
- **Node labels** appear only when zoom scale > 0.6 (below this threshold labels overlap and become illegible). This threshold is Claude's discretion to tune.
- **Tooltip text**: 12px (label size) — use `font-size: var(--font-size-label)` in the CSS tooltip overlay
- **Overlay controls**: 12px semibold (label pattern) matching existing `.filter-toggle` class

**Source:** Pre-existing tokens from `main.css` + RESEARCH.md Pattern 4 for Canvas 2D text rendering.

## Interaction Contract

Derived entirely from CONTEXT.md decisions. No ambiguity remains.

### Node interactions

| Interaction | Trigger | Result |
|-------------|---------|--------|
| Hover node | MouseMove — cursor within node radius | Show tooltip (title, first author, year) near cursor. No layout disruption. |
| Click node | MouseUp on node | Open Phase 8 paper drawer for this paper. Highlight direct neighbors + edges; dim everything else (0.25 alpha on non-connected elements). Graph stays visible under drawer. |
| Drag node | PointerDown + move on node | Pin node in place (freeze force layout for this node). Cursor changes to `grabbing`. |
| Click pinned node | MouseUp on already-pinned node | Unpin — resume force simulation for this node. |
| Hover edge | MouseMove — cursor within 4px of edge line | Show tooltip: regular edge "A cites B" with paper titles; special edge shows gap finding summary (shared terms, confidence %, justification snippet ≤80 chars). |

### Canvas interactions

| Interaction | Trigger | Result |
|-------------|---------|--------|
| Pan | PointerDown on background + move | Translate viewport offset. Cursor: `grab` during drag, `default` otherwise. |
| Zoom in/out | WheelEvent | Zoom toward cursor position. Scale range: 0.1× (fully zoomed out) to 4× (fully zoomed in). |
| Reset view | Double-click canvas background | Reset viewport to initial centered position, scale 1.0. |

### Overlay controls

| Control | Type | Behavior |
|---------|------|----------|
| "Contradiction" toggle | Filter toggle button | Show/hide contradiction edges. Active state: `--color-accent` background (matches `.filter-toggle.active` pattern). Default: visible (active). |
| "ABC-Bridge" toggle | Filter toggle button | Show/hide ABC-bridge edges. Active state: same pattern. Default: visible (active). |
| Play/Pause | Icon button | Toggle force simulation. Pause icon when running, play icon when paused. |
| Zoom in (+) | Icon button | Zoom in by 1.1× toward canvas center. |
| Zoom out (−) | Icon button | Zoom out by 0.9× toward canvas center. |

**Source:** CONTEXT.md interaction decisions + gap_card.rs filter toggle pattern.

## Layout Contract

### Graph page layout

The Graph page is a full-bleed canvas. The existing `.content-area` div already provides the container; the graph page must override its default `padding: var(--space-xl)` with `padding: 0` for the graph route only.

```
┌─ app-shell ───────────────────────────────────────────────┐
│ ┌─ sidebar ──┐  ┌─ content-area (padding: 0 for /graph) ─┐│
│ │  Dashboard │  │  ┌─ graph-page ───────────────────────┐ ││
│ │  Papers    │  │  │  <canvas> 100% × 100%              │ ││
│ │  Gaps      │  │  │  ┌─ graph-controls-overlay ──┐     │ ││
│ │  Problems  │  │  │  │  [Contradiction] [Bridge]  │     │ ││
│ │  Methods   │  │  │  │  [▶/⏸]  [+] [−]           │     │ ││
│ │▶ Graph     │  │  │  └────────────────────────────┘     │ ││
│ └────────────┘  │  └────────────────────────────────────┘ ││
│                 └────────────────────────────────────────┘ │
└────────────────────────────────────────────────────────────┘
```

- Sidebar stays visible (collapsible rail mode supported — graph canvas resizes)
- Paper drawer slides in from right over the canvas (z-index 100, same as existing drawer)
- Overlay controls: top-right corner, `position: absolute`, 16px from edges
- Canvas resize: on window resize and sidebar toggle, update `canvas.width` and `canvas.height` to match container pixel size

### Overlay controls layout

Two groups arranged vertically in top-right:

1. **Edge filter group**: `[Contradiction] [ABC-Bridge]` — horizontal row of filter toggles
2. **Simulation group**: `[▶/⏸] [+] [−]` — horizontal row of icon buttons

Each group has `background: var(--color-surface)`, `border: 1px solid var(--color-border)`, `border-radius: 6px`, `padding: 4px`.

## Edge Rendering Specification

Complete draw-order and style contract for the canvas renderer.

### Draw order (back to front)

1. Clear canvas with `--color-bg` (#0d1117)
2. Draw all regular citation edges (gray, 0.35 alpha, thin)
3. Draw contradiction edges (red, solid, 2px, full alpha)
4. Draw ABC-bridge edges (orange, dashed [6,4], 2px, full alpha)
5. Draw arrowheads on all visible edges
6. Draw all node circles (dimmed nodes drawn first if selection active)
7. Draw node labels (only if zoom scale > 0.6)
8. Draw highlighted node ring (accent color stroke, 2px, on selected node)

### Edge styles

| Edge type | Visible | Color | Width | Dash | Alpha |
|-----------|---------|-------|-------|------|-------|
| Regular citation | When "Contradiction" or "Bridge" toggled, regular edges stay visible | `#404040` | 1px | solid | 0.35 |
| Contradiction | Togglable, default on | `#f85149` | 2px | solid | 1.0 |
| ABC-bridge | Togglable, default on | `#d29922` | 2px | [6px, 4px] | 1.0 |

### Node styles

| State | Fill | Stroke | Radius formula |
|-------|------|--------|----------------|
| Default | `#4a9eff` | `#30363d` 1px | `max(4, min(18, sqrt(citation_count + 1) * 2.5))` px |
| Hovered | `#58a6ff` (brighter) | `#58a6ff` 2px | Same |
| Selected (clicked) | `#58a6ff` | `#58a6ff` 2px + outer ring 4px radius + | Same |
| Dimmed (non-neighbor) | `#2a3a4f` | `#30363d` 1px | Same |
| Pinned | `#4a9eff` + pin indicator (small `•` dot offset top-right at 8px) | `#30363d` 1px | Same |

## Animation & Transition Contract

| Element | Animation | Duration | Easing |
|---------|-----------|----------|--------|
| Force layout from random start | Continuous RAF until convergence (alpha < 0.001) | ~10s for 500 nodes | Barnes-Hut ticks, alpha decay 0.92/tick |
| Node settle (visual) | Continuous position interpolation from worker positions | Per-frame | Immediate — no CSS easing; physics controls smoothness |
| Paper drawer open/close | `transform: translateX` | 200ms | ease (pre-existing `--drawer-transition`) |
| Edge toggle show/hide | Immediate — re-render next RAF frame | 0ms | None (canvas repaint) |
| Play/pause button state | Immediate DOM class change | 0ms | None |
| Sidebar collapse/expand | Canvas resize + RAF resize call | 200ms transition | CSS handles sidebar; canvas resize syncs immediately |

## Accessibility Notes

- All overlay buttons have `aria-label` attributes (specified in copywriting contract above)
- Play/Pause button: `aria-pressed` attribute reflects simulation state (true when running)
- Edge toggle buttons: `aria-pressed` attribute reflects visibility state
- Canvas itself: `aria-label="Citation graph"` + `role="img"` (canvas content is not accessible to screen readers; the paper table on the Papers page serves as the accessible alternative)
- Keyboard navigation of canvas nodes is out of scope for Phase 9
