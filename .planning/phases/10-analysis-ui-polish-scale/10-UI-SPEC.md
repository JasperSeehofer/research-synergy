## Spacing Scale

Declared values (all multiples of 4 — from existing CSS custom properties):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (`--space-xs`) | Icon gaps, inline padding, heatmap cell gap |
| sm | 8px (`--space-sm`) | Compact element spacing, form elements |
| md | 16px (`--space-md`) | Default element spacing, card padding |
| lg | 24px (`--space-lg`) | Section padding, gap between cards |
| xl | 32px (`--space-xl`) | Layout gaps, drawer body/header padding |
| 2xl | 48px (`--space-2xl`) | Major section breaks |
| 3xl | 64px (`--space-3xl`) | Page-level spacing, empty state padding |

Exceptions:
- Touch targets (sidebar nav items, drawer close): min-height 44px — matches existing sidebar `.nav-item` pattern
- Drawer width: 480px (`--drawer-width`) — fixed layout value, not a spacing token
- Range slider thumb: 16px diameter — matches existing `input[type="range"]` styling
- Dual-handle range slider track height: 4px — matches existing `input[type="range"]` height

*Source: resyn-app/style/main.css `:root` block — all spacing tokens pre-exist.*

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0d1117` (`--color-bg`) | Main canvas background, graph background, empty states |
| Secondary (30%) | `#161b22` (`--color-surface`) / `#1c2128` (`--color-surface-raised`) | Sidebar, drawer background, cards, graph controls overlay, tab strip |
| Accent (10%) | `#58a6ff` (`--color-accent`) | Reserved list below |
| Destructive | `#f85149` (`--color-destructive` / `--color-contradiction`) | Contradiction edges, destructive actions only |

Accent (`#58a6ff`) reserved for:
1. Active navigation item border and text
2. Primary CTA buttons (`.btn-primary` background)
3. Focus ring on form inputs
4. Snippet highlight background overlay (`--color-accent-muted` #1f3a5f with accent text)
5. Active filter toggle (`.filter-toggle.active` background)
6. Active graph control button (`.graph-control-btn.active` background)
7. Node count indicator emphasis (none — muted text only, not accent)
8. Range slider thumb (both handles of the dual-range temporal slider)
9. Active drawer tab indicator (2px bottom border on active tab)
10. LOD-dimmed nodes: no accent — nodes at `alpha 0.03` lose color entirely

Additional semantic colors in use:
- Bridge/amber: `#d29922` (`--color-bridge`) — ABC-bridge graph edges
- Success/green: `#3fb950` (`--color-success`) — analyzed status, positive states
- Accent-muted: `#1f3a5f` (`--color-accent-muted`) — snippet highlight background, active nav background, table row hover

LOD and temporal opacity values (not color changes — alpha only):
- LOD hidden node: `alpha 0.03` (effectively invisible, topology trace only)
- Temporal out-of-range node: `alpha 0.10` — matches CONTEXT.md decision ("~10% opacity")
- Neighbor-dimmed node (existing Phase 9 pattern): `alpha 0.5`
- Combined (dimmed + temporal out-of-range): `alpha 0.5 * 0.10 = 0.05`

*Source: resyn-app/style/main.css `:root` — all color values pre-exist. Accent reserved-for list derived from CONTEXT.md decisions and existing CSS usage patterns.*

| Context | Heading | Body |
|---------|---------|------|
| Source tab — no extraction, paper analyzed | "No section text" | "This paper was analyzed before section extraction was available. Re-run analysis to populate source text." |

| Context | Copy |
|---------|------|

## Interaction Contract

### Provenance Flow (AUI-04)

1. User views gap findings on `/gaps` page
2. User clicks a paper ID button within a GapCard
3. Drawer opens to "Source" tab (not default "Overview")
4. The finding's `source_snippet` is passed to the drawer; the `source_section` determines which section to show first
5. The section text renders with the matching snippet span highlighted in `--color-accent-muted` background
6. User can switch to "Overview" tab to see abstract/findings — tab state is local to the current drawer open session
7. User can close drawer — tab state resets to "Overview" on next open

Transition behavior:
- Tab switching: immediate (no animation) — content swap only
- Drawer open: 200ms ease slide-in from right (existing `--drawer-transition`)
- Highlight: no animation — static background color

### LOD (SCALE-02)

Progressive reveal is driven by `viewport.scale` (zoom level), evaluated each RAF frame:

| Zoom (viewport.scale) | Nodes visible |
|----------------------|---------------|
| < 0.3 | Seed paper + depth-1 direct references only |
| 0.3 – 0.6 | + citation_count >= 50 (high-citation nodes) |
| 0.6 – 1.0 | + citation_count >= 10 and depth <= 2 |
| > 1.0 | All nodes |

Hidden nodes:
- Node itself: `alpha 0.0` (not drawn, zero-alpha — skip draw call or draw at 0)
- Hidden node's edges: `alpha 0.05` (faint topology trace — renders as barely-visible gray line)
- Transition: opacity fade over 150ms (lerp alpha toward target each frame at rate 0.15 per frame) — not pop-in

Node count indicator updates in real time as zoom changes.

### Temporal Filter (SCALE-03)

- Slider handles drag in real-time; `on:input` fires on every pointer move
- Nodes outside selected year range: `alpha 0.10` (dimmed, not hidden)
- Nodes with unknown year (year = 0 or parse fails): treated as in-range (always visible at full alpha)
- Analysis panels (gaps, open problems, methods pages): no filtering — always show full corpus
- Year range display updates in real time alongside slider drag
- Initial state: full range (min_year to max_year of loaded graph data)

Alpha composition (when multiple visibility states apply simultaneously):
```
final_alpha = base_alpha × lod_alpha × temporal_alpha
```
Where:
- `base_alpha`: 0.5 if neighbor-dimmed (Phase 9), else 1.0
- `lod_alpha`: 0.0 if LOD-hidden (edges 0.05), else 1.0
- `temporal_alpha`: 0.10 if outside year range, else 1.0

*Source: CONTEXT.md (temporal ~10% opacity, real-time drag, LOD fade). RESEARCH.md (exact threshold values, alpha composition formula, on:input pattern).*
