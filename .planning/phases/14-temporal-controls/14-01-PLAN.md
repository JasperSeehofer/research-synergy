Edit `resyn-app/src/components/graph_controls.rs` in the `TemporalSlider` component to add value clamping (per D-05):

1. **Min input handler (line ~100):** Change `temporal_min.set(val);` to `temporal_min.set(val.min(temporal_max.get_untracked()));`. This prevents min from exceeding max.
