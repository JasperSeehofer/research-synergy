**Step 1: Create `resyn-app/src/components/analysis_controls.rs`**

This component renders:
1. **LLM Warning Banner** (per D-07): shown when no LLM provider is configured. The server needs to communicate this state. Create a simple server function `check_llm_configured` that returns `bool` based on `std::env::var("RESYN_LLM_PROVIDER").is_ok()`. Place it in the same file.

2. **Run Analysis Button** (per D-01): a `btn-primary` button that calls `start_analysis()`. Disables while analysis SSE events are active. Label: "Run Analysis" (idle) / "Analysis running..." (pending, disabled).

The component subscribes to the SSE `/progress` endpoint to track analysis state. It needs `last_event` as a prop OR subscribes itself. Since `CrawlProgress` already subscribes to SSE, to avoid duplicate subscriptions, accept the analysis state as a prop signal.

**Better approach:** Create `AnalysisControls` as a standalone component that accepts a `RwSignal<Option<ProgressEvent>>` prop for the last SSE event (shared with CrawlProgress from a parent). OR simply have it use its own SSE subscription — multiple subscribers to the same broadcast channel is fine in Leptos (each `use_event_source` creates its own EventSource connection which is lightweight).

Use its own SSE subscription for simplicity:

```rust
use codee::string::JsonSerdeCodec;
use leptos::prelude::*;
use leptos_use::{UseEventSourceReturn, use_event_source};
use resyn_core::datamodels::progress::ProgressEvent;

use crate::server_fns::analysis::start_analysis;

/// Check if LLM provider is configured (server-side env var check).
#[server(CheckLlmConfigured, "/api")]
pub async fn check_llm_configured() -> Result<bool, ServerFnError> {
    #[cfg(feature = "ssr")]
    {
        Ok(std::env::var("RESYN_LLM_PROVIDER").is_ok())
    }
    #[cfg(not(feature = "ssr"))]
    unreachable!()
}

/// Analysis controls: LLM warning banner + Run Analysis button.
#[component]
pub fn AnalysisControls() -> impl IntoView {
    let llm_configured = Resource::new(|| (), |_| check_llm_configured());

    let UseEventSourceReturn { message: sse_message, .. } =
        use_event_source::<ProgressEvent, JsonSerdeCodec>("/progress");

    let last_event: RwSignal<Option<ProgressEvent>> = RwSignal::new(None);
    Effect::new(move |_| {
        if let Some(msg) = sse_message.get() {
            last_event.set(Some(msg.data));
        }
    });

    let is_analysis_running = move || {
        last_event.get().as_ref()
            .map(|e| e.event_type.starts_with("analysis_") && e.event_type != "analysis_complete" && e.event_type != "analysis_error")
            .unwrap_or(false)
    };

    let analysis_action = Action::new(move |_: &()| async move {
        start_analysis().await
    });

    let is_pending = analysis_action.pending();

    view! {
        // LLM Warning Banner (D-07)
        <Suspense fallback=|| ()>
            {move || llm_configured.get().map(|result| {
                match result {
                    Ok(false) => view! {
                        <div class="warning-banner">
                            "LLM provider not configured \u{2014} showing NLP-only results. Set RESYN_LLM_PROVIDER for full analysis."
                        </div>
                    }.into_any(),
                    _ => view! { <span></span> }.into_any(),
                }
            })}
        </Suspense>

        // Run Analysis Button (D-01)
        <div style="margin-top: var(--space-lg);">
            <button
                class="btn-primary"
                disabled=move || is_pending.get() || is_analysis_running()
                on:click=move |_| analysis_action.dispatch(())
            >
                {move || if is_pending.get() || is_analysis_running() {
                    "Analysis running\u{2026}"
                } else {
                    "Run Analysis"
                }}
            </button>
        </div>

        // Result feedback
        {move || analysis_action.value().get().map(|r| match r {
            Ok(msg) => view! {
                <p class="status-text success">{msg}</p>
            }.into_any(),
            Err(e) => view! {
                <p class="status-text error">{format!("Error: {}", e)}</p>
            }.into_any(),
        })}
    }
}
```
