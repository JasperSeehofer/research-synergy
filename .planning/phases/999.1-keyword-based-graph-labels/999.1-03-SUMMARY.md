# Phase 999.1 Plan 03: Label Mode Dropdown and Keyword Pill Rendering Summary

Label mode dropdown added to graph controls with Keywords / Author-Year / Off options; per-node keyword pill rendering with score-based opacity and unanalyzed fallback badge implemented in Canvas 2D overlay.

## What Was Built

### Task 1: label_collision.rs â€” opacity, draw_keyword_pills, draw_not_analyzed_badge

- `draw_label_pill` extended with `opacity: f64` parameter. Sets `set_global_alpha(opacity)` before drawing and `set_global_alpha(1.0)` after. All existing callers updated to pass `1.0`.
- `PILL_GAP: f64 = 4.0` constant added for horizontal spacing between keyword pills.
- `build_text_widths` now takes `label_mode: &LabelMode` parameter:
  - `AuthorYear`: measures `node.label()` (existing behavior)
  - `Off`: returns `vec![0.0; nodes.len()]` (no labels)
  - `Keywords`: measures "[not analyzed]" for empty `top_keywords`, or sums top-2 keyword text widths + 2 * PILL_H_PAD per pill + PILL_GAP between pills
- `draw_keyword_pills()`: takes center x, label_y, keywords slice, and pre-measured per-pill text widths. Lays out top-2 pills horizontally centered, each with `opacity = score_to_opacity(score)` and `bg_opacity = 0.85 * opacity`. Uses `rgba(22, 27, 34, {bg_opacity})` background, `rgba(48, 54, 61, 0.6)` border, `#e6edf3` text.
- `draw_not_analyzed_badge()`: single pill with "[not analyzed]" text, `rgba(22, 27, 34, 0.5)` background, `rgba(48, 54, 61, 0.4)` border, `#8b949e` muted text.

### Task 2: graph_controls.rs + graph.rs â€” dropdown and RAF branching

**graph_controls.rs:**
- Added `use crate::graph::layout_state::LabelMode` import.
- Added `label_mode: RwSignal<LabelMode>` parameter to `GraphControls`.
- Added fourth `graph-controls-group` div with a `<select>` offering "Author / Year", "Keywords", "Off" options. `on:change` handler maps select value string to `LabelMode` variant.

**graph.rs:**
- Added `use crate::graph::layout_state::LabelMode` import.
- Added `let label_mode: RwSignal<LabelMode> = RwSignal::new(LabelMode::AuthorYear)` near other signals.
- `RenderState` extended with `keyword_text_widths: Vec<f64>` and `prev_label_mode: LabelMode`.
- Text width measurement at graph load now computes both `text_widths` (AuthorYear) and `keyword_text_widths` (Keywords) from the same temporary canvas context.
- `start_render_loop` receives `label_mode: RwSignal<LabelMode>`.
- RAF loop: before cache rebuild, reads `label_mode.get_untracked()`, writes to `s.graph.label_mode`, detects change vs `s.prev_label_mode` and sets `label_cache_dirty = true`.
- Cache rebuild uses `keyword_text_widths` for Keywords mode, `text_widths` for others.
- Label drawing section replaced with 3-way `match s.graph.label_mode`:
  - `Off`: hover node label only (author-year pill at opacity 1.0)
  - `AuthorYear`: collision-culled cache draw + hover fallback (existing behavior, now with opacity 1.0)
  - `Keywords`: collision-culled cache, draws `draw_not_analyzed_badge` or `draw_keyword_pills` per node; inline `measure_text` for per-pill widths
- `GraphControls` invocation updated to pass `label_mode=label_mode`.
