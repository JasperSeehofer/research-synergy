## Spacing Scale

Declared values (multiples of 4 — pulled from `--space-*` tokens in `main.css`):

| Token | CSS Variable | Value | Usage |
|-------|-------------|-------|-------|
| xs | `--space-xs` | 4px | Icon gaps, pill inter-gap |
| sm | `--space-sm` | 8px | Pill horizontal padding, control padding |
| md | `--space-md` | 16px | Default element spacing |
| lg | `--space-lg` | 24px | Section padding |
| xl | `--space-xl` | 32px | Layout gaps |
| 2xl | `--space-2xl` | 48px | Major section breaks |
| 3xl | `--space-3xl` | 64px | Page-level spacing |

Exceptions:
- Keyword pill inter-gap: 4px (`--space-xs`) between adjacent pills for the same node
- Canvas-rendered pill horizontal padding: 8px (`PILL_H_PAD = 8.0` constant in `label_collision.rs`) — matches `--space-sm`
- Graph control button minimum width: 32px height (existing `.graph-control-btn` constraint)
- Convex hull dashed border padding: 12px outset from outermost hull node center — between `--space-sm` and `--space-md`, chosen to avoid overlapping node circles

### Score-Based Opacity Mapping (D-12, Claude's Discretion)

TF-IDF scores are normalized in `[0, 1]`. Opacity mapping:

| Score | Opacity |
|-------|---------|
| 0.0 | 0.35 (minimum — pill always visible but clearly secondary) |
| 0.5 | 0.67 (midpoint, linear interpolation) |
| 1.0 | 1.0 (maximum prominence) |

Formula: `opacity = 0.35 + (score * 0.65)`

This gives every pill a minimum opacity of 0.35 (readable at any zoom) and reserves full opacity for top-scoring keywords. Applies to both pill background (`set_global_alpha`) and is restored to 1.0 after each pill draw.

## Interaction Contract

### Label Mode Dropdown (D-01)

| Property | Value |
|----------|-------|
| Element | `<select>` — uses existing CSS `select` and `.form-select` styles from `main.css` |
| Placement | `graph-controls-overlay`, new fourth group below simulation controls |
| Label | `"Label mode"` (`12px`, `--color-text-muted`, uppercase — matching `.sidebar-title` pattern) |
| Default selected | `"Author / Year"` — preserves existing behavior on initial load |
| Options | `Author / Year`, `Keywords`, `Off` |
| Width | `fit-content`, min 120px to match existing button group width |
| On change | Updates `RwSignal<LabelMode>` immediately; triggers `label_cache_dirty = true` to re-measure text widths |

### Hover Tooltip Extension (D-05)

Extended tooltip layout (appended below existing title/author/citation line):

```
{title truncated to 60 chars}
{first_author} · {year} · {citation_count} citations
─────────────────────────
Keywords:
  {term_1}  {score_1:.2}
  {term_2}  {score_2:.2}
  {term_3}  {score_3:.2}
  {term_4}  {score_4:.2}
  {term_5}  {score_5:.2}
```

- Section separator: em dash line (`────────`) drawn as text using `--color-border` color
- Keyword lines: 2-space indent, term left-aligned, score right-aligned (monospace makes alignment natural)
- Font: existing tooltip CSS (`--font-size-label: 12px`, `line-height: 1.3`) unchanged
- Max-width: `280px` (existing `.graph-tooltip` constraint)
- Only shown when `LabelMode::Keywords` is active AND `node.top_keywords` is non-empty
- Fallback for unanalyzed nodes: no "Keywords:" section shown; tooltip shows title/author/citations only

### Cluster Label LOD Transition (D-08, Claude's Discretion)

| Zoom scale | Cluster labels | Per-node labels |
|------------|---------------|-----------------|
| `< 0.6` (LOD_LEVEL_1) | Visible | Collision-culled (sparse at this zoom) |
| `>= 0.6` | Hidden | Visible (per-node pills take over) |

Transition is instant (no fade animation) — consistent with existing LOD system.

### K-Means Recompute Schedule (D-09, Claude's Discretion)

| Condition | Action |
|-----------|--------|
| `simulation_settled == true` (first settle) | Full recompute |
| `simulation_running == true` | Recompute every 10 RAF frames (~6x/second at 60fps) |
| `simulation_running == false` | No recompute (positions stable) |
| `label_mode != Keywords` | Skip clustering entirely |

### Cluster Count Heuristic (D-07, Claude's Discretion)

`k = clamp(floor(sqrt(n / 2)), 3, 8)` where n = total visible node count.

| Node count | k |
|------------|---|
| < 18 | 3 (minimum) |
| 18–31 | 3–3 |
| 32–49 | 4 |
| 50–71 | 5 |
| 72–97 | 6 |
| 98–127 | 7 |
| >= 128 | 8 (maximum) |

### Convex Hull Style (D-10, Claude's Discretion)

| Property | Value |
|----------|-------|
| Line color | `rgba(88, 166, 255, 0.25)` — accent at 25% opacity |
| Line width | 1.5px |
| Dash pattern | `[4, 4]` — 4px on, 4px off |
| Outset padding | 12px beyond outermost node center in screen space |
| Fill | `rgba(88, 166, 255, 0.04)` — extremely subtle tint (region hint, not fill) |

### "[not analyzed]" Badge (D-06)

| Property | Value |
|----------|-------|
| Text | `[not analyzed]` |
| Font | 11px monospace |
| Background | `rgba(22, 27, 34, 0.5)` |
