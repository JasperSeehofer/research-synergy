## Ring Rendering Approach

### Q1: Where should topic ring arcs be placed relative to existing node border?

| Option | Description | Selected |
|--------|-------------|----------|
| Replace node border | Topic arcs replace existing 1px border stroke. Border IS the topic ring. Seed keeps planetary ring. | ✓ |
| Outer ring layer | Keep existing border, add topic arcs as separate outer ring. More visual weight. | |
| You decide | Claude picks. | |

**User's choice:** Replace node border (Recommended)
**Notes:** Clean approach, no extra visual layer.

### Q2: WebGL2 renderer implementation for topic arcs?

| Option | Description | Selected |
|--------|-------------|----------|
| Canvas2D overlay for arcs | Draw topic arcs on Canvas2D overlay. No shader changes. Both renderers share overlay. | ✓ |

| You decide | Claude picks. | |

**User's choice:** Canvas2D overlay for arcs (Recommended)
**Notes:** Avoids complex shader changes.

### Q3: Should topic rings be visible in all label modes?
